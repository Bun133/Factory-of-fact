# Factory-of-fact
このゲームの趣旨は、MindustryとfactorioとASTRONEERを混ぜ混ぜする感じです。

初期版は2D

factorioのように自動製作機や電車を取り入れ、
Mindustryのようなウェーブ制、タレットを取り入れ、
ASTRONNERのようなオープンワールド性を取り入れていくつもりです。
